# Mohamed ALAOUI - DJ Portfolio

A creative, modern portfolio website for DJ Mohamed ALAOUI featuring Moroccan-inspired design elements.

## Features
- 🎵 Animated spinning vinyl disc in header
- 🇲🇦 Moroccan geometric patterns and gold/copper color palette
- 📱 Fully responsive design
- ✨ Smooth scroll animations
- 🎨 Dark theme with elegant typography

## File Structure
```
├── index.html      # Main HTML structure
├── style.css       # Styling with CSS custom properties
├── app.js          # Interactivity and animations
└── README.md       # Documentation
```
